Pre-Deployment Essentials

To run the NMT suite, you must have the following environment prepared:

Software Requirements

  • OS: Windows 10 / 11 (64-bit)
  • Python: Version 3.10+ (Check 'Add to PATH' during install)
  • OCR Engine: Tesseract OCR installed on your system
  • Permissions: Administrator access for Taskbar pinning

Source Components

The installer will fetch these files from the repository:

  • NMT.py (The Main Engine)
  • nmt-u-c.py (The Self-Building Updater)
  • version.uvt (Release Tracker)

Official Execution Guide

Follow these steps to deploy the DarkFox Co. environment correctly:

Step 1: Handshake

Download the NMT-Installer.py from the repository link below. Place it in a dedicated folder.

Step 2: Dependency Injection

Run the installer. It will automatically detect your environment and install customtkinter, pyautogui, pytesseract, and pyinstaller.

Step 3: Binary Forge

The installer downloads the raw source code and compiles it locally. NMT.exe and nmt-u-c.exe are generated specifically for your hardware.

Step 4: Integration

Once the build is complete, check your Desktop for the NMT Shortcut and your Taskbar for the pinned icon. Deployment is finished.
