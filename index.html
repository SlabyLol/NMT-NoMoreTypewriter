Deployment Guide

  • Execute the NMT-Installer.py to initiate the environment handshake.
  • The system will automatically pull dependencies and generate the DarkFox identity assets (Icons/Shortcuts).
  • Local compilation via PyInstaller ensures a hardware-specific, high-performance binary.
  • Launch the environment and select your target window from the dynamic process list.
